# Productivity Dashboard

A personal productivity dashboard to manage daily tasks, goals, habits, and stay motivated.

## Features
- **Todo List**: Manage daily tasks efficiently.
- **Daily Planner**: Schedule your day hour by hour.
- **Goals**: Set and track daily goals.
- **Pomodoro Timer**: Focus on work with intervals.
- **Motivational Quotes**: Get inspired with random quotes.
- **Habit Tracker**: Track progress on daily habits.

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
